API

概述

API（Application Programing Interface 应用程序编程接口）。
Java API是Java程序员的词典，JDK中提供给我们使用的类的说明文档。这些将底层代码实现封装了起来，我们不需要关心这些类是如何实现的，只需要学习这些类如何使用即可。所以我们可以通过查询API的方式，学习Java提供的类，并且得知如何使用它们。

API使用步骤

  1. 打开帮助文档
  2. 点击显示，打开索引
  3. 在输入框中输入
  4. 看包，Java.lang下的类不需要导包，其他的需要
  5. 看类的解释和说明
  6. 学习构造方法
  7. 使用成员变量
