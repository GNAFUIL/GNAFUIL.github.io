ArrayList类

数组的长度不可以发生改变
但是ArrayList集合的长度是可以随意变化的
对于Arraylist来说，有一个<E>代表泛型
泛型：也就是装在集合当中的所有元素，都是统一的类型(都是String或者都是Person)
泛型只能是引用类型，不能是基本类型（两个浮点型的，float和double；四个整型的：byte，short，int，long；一个字符类型：char，一个boolean型的）

实例

注意事项

对于ArrayList集合来说，直接打印得到的不是地址值，而是内容
如果内容是空，那么得到的是空的中括号：[]

ArrayList常用方法

ArrayList当中常见的方法有：

  1. public boolean add(E e);
  2. public E get(int index);
  3. public E remove(int index);
  4. public int size();

public boolean add(E e); 向集合当中添加元素，参数类型和泛型一致。
备注：对于ArrayList集合来说，add添加动作一定是成功的，所以返回值可用可不用
但是对于以后学习的其他集合来说，add添加动作不一定成功，所以返回值代表添加是否成功

public E get(int index); 从集合当中获取元素，参数是索引编号，返回值就是对应位置的元素

public E remove(int index); 从集合当中删除元素，参数是索引编号，返回值就是被删除掉的元素

public int size(); 获取集合的尺寸长度，返回值时集合当中包含的元素个数

使用ArrayList集合储存基本数据类型

如果希望向集合ArrayList当中储存基本类型数据，必须使用基本类型对应的包装类
包装类：将目标数据包装成一个对象。

基本类型 包装类(引用类型，包装类都位于java.lang包里)
byte Byte
short Short
int Integer [特殊]
long Long
float Float
double Double
char Character [特殊]
boolean Boolean

从JDK 1.5以后，支持自动装箱、自动拆箱

自动装箱：基本类型—->包装类型
自动拆箱：包装类型—->基本类型

练习

题目一

生成6个1~33之间的随机整数，添加到集合，并遍历集合。
思路：

  1. 生成6个随机整数
  2. add到ArrayList集合中
  3. list.fori遍历输出

题目二

自定义4个学生类，添加到集合当中，遍历集合

思路：

  1. 创建4个学生类，创建ArrayList<Student>集合
  2. 将创建的学生类添加到集合当中
  3. 遍历输出

Student类

注意事项

如果学生类没有private，则不需要创建getter setter方法
如果经过private处理，则需要额外创建getter setter方法，并且遍历输出是需要调用getter方法

题目三

定义以指定格式打印集合的方法(ArrayList类型作为参数)，使用{}括起来，使用@分割每个元素。
例如：{甲@乙@丙@丁}

题目四

使用一个大集合存入20个随机数字，然后筛选其中的偶数元素，放到小集合中
要求使用自定义方法来实现筛选

思路

  1. 创建集合储存int数字：<Integer>
  2. Random nextInt创建随机数字
  3. for循环add放入
  4. 定义方法，筛选
