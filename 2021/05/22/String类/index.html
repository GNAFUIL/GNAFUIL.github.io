String类

java.lang.String类代表字符串
程序中所有双引号字符串都是String类型的对象

字符串的特点

  1. 字符串的内容创建后不可变;字符串是常量，它的值在创建之后不能更改
  2. 因为字符串不可改变，所以字符串是可以共享使用的;
  3. 字符串效果上相当是char[]字符数组，但是底层原理是byte[]字节数组

创建字符串的常见方法

常见方法有3+1种方式

三种构造方法:

public String():创建一个空白字符串，不含有任何内容
public String(char[] array):根据字符数组的内容，创建对应的字符串
public String(byte[] array):根据字节数组的内容，创建对应的字符串

一种直接创建:

String str="Hello";

实例

注意

直接写上双引号就是字符串对象(没有new一样是字符串对象)

字符串的常量池

程序当中直接写上的双引号字符串，就在（堆当中的）字符串常量池中。
程序当中直接写上的双引号字符串，将会在常量池中创建一个字符串对象，而new的不在池当中
字符串对象保存的也是一个地址值，这个地址值指向的是堆当中使用String底层本质的byte[]数组
当常量池中已经有需要的内容时，不会再创建新的，而是直接引用

实例

此Demo中str1已经在常量池中创建了指向代表”abc“的byte[](97，98，99)数组的字符串对象
当str2再想创建时相同含义的字符串对象时，就不再创建而是变为直接在常量池中引用字符串对象

对于char[] chars={'a','b','c'},代表在栈当中创建一个char[] chars
其中保存着指向对当中代表着'a','b','c'的char[]('a','b','c')数组的地址值

String str3=new String(chars)
String str3代表在栈当中创建一个String str3
new String(chars)
new代表着在堆中创建字符串对象，就与字符串常量池无关
String(chars),指将chars中地址值指向的char[]('a','b','c')数组转化为byte[](97，98，99)数组

所以此语句的含义是，在栈中创建一个str3，它保存着指向堆当中新建的但不在字符串常量池的字符串对象的地址
字符串对象保存着指向将char[]('a','b','c')数组转化为`byte数组的地址

注意事项

  1. 对于基本类型来说，== 是进行数值的比较
  2. 对于引用类型来说，== 是进行地址值的比较
  3. 双引号直接写的字符串在常量池当中，new的不在池当中

字符串的常用方法

字符串比较的相关方法

== 是进行对象的地址比较
如果确实需要字符串的内容比较，可以使用两个方法

1. public boolean equals(Object obj)

参数可以是任何对象，只有参数是一个字符串并且内容相同的才会给一个true,否则返回false

代码
注意事项:
  1. 任何对象都能用Object进行接收。
  2. equals方法具有对称性，也就是a.equals(b)与b.equals(a)效果一样(对象一和对象二比较和对象二与对象一比较能不一样吗。。。)
  3. 如果比较双方一个常量一个变量，推荐把常量字符串写在前面
    推荐:"abc".equals(str)
    不推荐:str.equals("abc")

2. public boolean equalsIgnoreCase(String str)

忽略大小写进行比较

实例

字符串获取的相关方法

常见方法有

1. public int length()

获取字符串当中含有的字符个数(字符串长度)

2. public String concat(String str)

将当前字符串和参数字符串拼接成为返回值新的字符串

3. public char chaAt(int index)

获取指定索引位置的单个字符。(索引从0开始)

4. public int index(String str)

查找参数字符串在本字符串当中首次出现的索引位置，如果没有返回-1值。

字符串截取方法

1. public String substring(int index)

截取从索引地址位置一直到字符串末尾，返回新字符串

2. public String substring(int begin,int end)

截取从begin索引地址位置一直到end索引地址结束的中间字符串，返回新字符串 [begin,end),包含左边，不包含右边

注意事项

下面这种写法字符串的内容没有发生改变，改变的是strA保存指向字符串常量池的地址值
下面有两个字符串:"Hello","World"
strA当中的保存的是地址值
本来的地址值是指向Hello的，后来的地址值是指向World的
两字符串没有改变，改变的是strA保存的地址值

字符串转换的相关方法

1. public char[] toCharArray()

将当前字符串拆分成为字符串数组作为返回值

2. public byte[] getBytes()

获得当前字符串底层的字节数组

3. public String replace(CharSequence^1 oldString,CharSequence^1 newString)

将所有出现的老字符串替换成新字符串，返回替换之后的结果新字符串

字符串的分割方法

public String[] split(String regex)

按照参数的规则，将字符串切分成若干部分

注意事项

split方法的参数其实是一个正则表达式,如果安装英文句点”.“进行切分，必须写”\\.“加上两个反斜杠
